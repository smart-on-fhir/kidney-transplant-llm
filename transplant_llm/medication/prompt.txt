# Task: Extract MedicationRequest-like episodes (Kidney Transplant)

Output JSON that validates against `ChartReviewBatch.outputs[*]` (see `python/llm_models/schemas_mr.py`).

Rules:
1) Normalize ingredient to generic; map class (CNI, ANTIMET, STEROID, MTOR, BELATACEPT, INDUCTION, B_CELL, IVIG, PLEX).
2) Intent:
   - INDUCTION: agents within POD −1..+7 or explicitly labeled induction (ATG, basiliximab, IV methylpred pulses).
   - MAINTENANCE: ongoing outpatient CNI/ANTIMET/STEROID; belatacept; mTOR.
   - RESCUE: rejection/desensitization (IVIG, rituximab, plasmapheresis, steroid pulses).
3) Capture dose (value+unit), route, normalized frequency (QD/BID/TID/Q4W), authored_on if visible.
4) Dates: start_date = earliest specific date you can cite; stop_date if explicitly ended/held.
5) If days supply or repeats are stated, record them. Otherwise null.
6) Provide ≥1 evidence snippet per episode with note_date and exact text.
7) If ambiguous, set certainty=LOW and include rationale in snippet context.
